## 轨道执行器
- 轨道执行器用于执行轨道文件所记录的刷图操作。其中<span style="color: red">普通任务</span>用于单个关卡的重复刷图（如 100 局神殿）；<span style="color: red">高级任务</span>则可以连刷多个关卡（如悬赏+勇士+公会任务），且拥有定时启动、自动刷新等强大功能。
